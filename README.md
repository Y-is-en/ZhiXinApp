# 2. 项目包含模块如下

>+ App引导页、欢迎页
>+ 登录注册 (基于Bmob后端云数据库) / 基于一登的刷脸登陆
>+ 新闻 (数据源于网易新闻）
>+ 天气（仿造欧酷天气）
>+ 图灵机器人 (基于图灵机器人和讯飞语音识别技术）
>+ 开发者个人信息
>+ 蒲公英应用发布平台及应用自动更新功能
